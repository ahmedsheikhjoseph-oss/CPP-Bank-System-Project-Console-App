# Bank Management System (Level 1) 🏦

## 📝 Overview
This project is a console-based **Bank Management System** developed as the first major project in **Algorithms & Problem-Solving Level 3**. It demonstrates advanced control over data persistence using File Handling and structured programming.

## 🚀 Key Features
- **Client Management:** Create, Read, Update, and Delete (CRUD) client records.
- **Data Persistence:** All data is stored and retrieved from a `.txt` database.
- **Search & Verification:** Search for clients by account number with real-time validation.
- **Formatted UI:** Clean table views using `iomanip` for professional data display.

## 🛠️ Technical Concepts Used
- **File Handling:** `fstream` (Read/Write/Append).
- **Data Structures:** `vector` and `struct` to manage complex client records.
- **String Manipulation:** Custom string splitting and record parsing.
- **Logic:** Input validation and account verification.

## 📖 How to Use
1. Ensure you have a `Clients.txt` file in the same directory.
2. Compile and run the code.
3. Use the Main Menu to navigate through system functions.
