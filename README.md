# Bank Management System (Level 1) 🏦

## 📝 Overview
This project is a console-based **Bank Management System** developed as the first major project in **Algorithms & Problem-Solving Level 3**. It demonstrates advanced control over data persistence using File Handling and structured programming.

## 🚀 Key Features
- **Client Management:** Create, Read, Update, and Delete (CRUD) client records.
- **Data Persistence:** All data is stored and retrieved from a `.txt` database.
- **Search & Verification:** Search for clients by account number with real-time validation.
- **Formatted UI:** Clean table views using `iomanip` for professional data display.

## 🛠️ Technical Concepts Used
- **File Handling:** `fstream` (Read/Write/Append).
- **Data Structures:** `vector` and `struct` to manage complex client records.
- **String Manipulation:** Custom string splitting and record parsing.
- **Logic:** Input validation and account verification.

## 📖 How to Use
1. Ensure you have a `Clients.txt` file in the same directory.
2. Compile and run the code.
3. Use the Main Menu to navigate through system functions.

# نظام إدارة البنك (المستوى الأول) 🏦

## 📝 نظرة عامة
هذا المشروع عبارة عن **نظام إدارة بنكي** يعمل من خلال شاشة وحدة التحكم (Console)، تم تطويره كأول مشروع رئيسي في دورة **الخوارزميات وحل المشكلات - المستوى الثالث**. يبرز المشروع القدرة على التحكم المتقدم في حفظ البيانات واسترجاعها باستخدام أنظمة الملفات والبرمجة الهيكلية.

## 🚀 المميزات الرئيسية
- **إدارة العملاء:** إجراء العمليات الأساسية (CRUD) مثل: إضافة، عرض، تعديل، وحذف بيانات العملاء.
- **حفظ البيانات:** يتم تخزين واسترجاع كافة البيانات من قاعدة بيانات نصية بصيغة `.txt`.
- **البحث والتحقق:** ميزة البحث عن العملاء برقم الحساب مع التحقق الفوري من صحة البيانات.
- **واجهة مستخدم منظمة:** عرض البيانات في جداول احترافية باستخدام مكتبة `iomanip`.

## 🛠️ المفاهيم التقنية المستخدمة
- **التعامل مع الملفات (File Handling):** استخدام `fstream` للقراءة والكتابة والإضافة.
- **هياكل البيانات:** استخدام الـ `vector` والـ `struct` لإدارة سجلات العملاء المعقدة.
- **معالجة النصوص:** وظائف مخصصة لتقسيم النصوص (String Splitting) وتحليل السجلات.
- **المنطق البرمجي:** التحقق من المدخلات والتأكد من عدم تكرار أرقام الحسابات.

## 📖 كيفية الاستخدام
1. تأكد من وجود ملف باسم `Clients.txt` في نفس مجلد المشروع.
2. قم بتجميع (Compile) وتشغيل الكود.
3. استخدم القائمة الرئيسية للتنقل بين وظائف النظام المختلفة.
